# 📊 Stock-Based Financial Account Management System

A comprehensive financial account management platform built with **Java Spring Boot** and **PostgreSQL**, designed to help financial advisors, portfolio managers, and investment firms track, analyze, and optimize client investments based on stock market data.

---

## 🚀 Features

### Client Account Management
- Create and manage multiple client profiles
- Track individual investment goals and risk profiles
- Maintain transaction history

### Market Data Integration
- Fetch live and historical stock prices
- Support for multiple stock exchanges
- Automated price updates

### Security & Access Control
- Role-based access (admin, advisor, client)
- Encrypted data storage
- Secure authentication

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot  
- **Database:** PostgreSQL  
- **Authentication:** JWT  
- **Build Tool:** Maven
